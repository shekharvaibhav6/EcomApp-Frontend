# 🛒 EcommerceApp Frontend

A modern and responsive E-Commerce Web Application built with React.js that delivers a seamless online shopping experience.

This frontend communicates with the Spring Boot backend through REST APIs and provides a fast, intuitive, and mobile-friendly user interface for customers.

---

## ✨ Features

### 👤 Authentication
- User Registration
- User Login
- JWT-based Authentication
- Protected Routes
- Persistent Login Sessions

### 🛍️ Shopping Experience
- Browse Products
- Product Details Page
- Search Products
- Category-wise Products
- Add to Cart
- Update Cart Quantity
- Remove Items from Cart

### 📦 Order Management
- Place Orders
- View Order History
- Order Summary

### 🎨 User Interface
- Fully Responsive Design
- Modern React Components
- Clean Navigation
- Mobile-Friendly Layout
- Reusable UI Components

🚀 Getting Started
Clone Repository
git clone https://github.com/shekharvaibhav6/EcomApp-Frontend


Navigate to Project

cd EcommerceApp-Frontend
Install Dependencies
npm install
Run Application
npm start

Application runs on:

http://localhost:3000
🔗 Backend Configuration

Update the API Base URL according to your backend server:

const API_BASE_URL = "http://localhost:8080/api";
